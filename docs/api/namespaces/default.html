get_value()

get_value(string  $value_name, string  $filter = "string") : string|null

La fonction *get_value* permet de récupérer une valeur contenue dans *$_POST*, la sanitise et la retourne.

Retourn null s'il y a une erreur.

Parameters

string $value_name

La clef de la valeur que l'on souhaite récupérer
string $filter

Le type de filtre que l'on souhaite utiliser pour la validation/sanitisation (valeurs acceptées : string, email)
